## Install

<CodeGroup>

```bash pnpm
pnpm add @slicekit/core @wagmi/core viem ethers@5.7.2 merkletreejs
```

```bash bun
bun add @slicekit/core @wagmi/core viem ethers@5.7.2 merkletreejs
```

```bash npm
npm install @slicekit/core @wagmi/core viem ethers@5.7.2 merkletreejs
```

```bash yarn
yarn add @slicekit/core @wagmi/core viem ethers@5.7.2 merkletreejs
```

</CodeGroup>
