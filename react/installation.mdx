## Install

<CodeGroup>

```bash pnpm
pnpm add @slicekit/react @wagmi/core viem
```

```bash bun
bun add @slicekit/react @wagmi/core viem
```

```bash npm
npm install @slicekit/react @wagmi/core viem
```

```bash yarn
yarn add @slicekit/react @wagmi/core viem
```

</CodeGroup>
